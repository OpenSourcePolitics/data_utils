Scripts d'extraction et de visualisation de données.
Pour utiliser les scripts, vous pouvez utiliser un notebook jupyter avec le module jupytext.

## Setup

- Créer un environnement virtuel dans ce dossier, et l'activer
- installer toutes les libraries définies dans le pyproject.toml (par exemple, avec `pip install -e .`)
- lancer `jupyter lab`, puis ouvrir les scripts avec click droit puis "open with notebook"
